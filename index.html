Research & Thesis

🫀 Design of ECG Signal Quality Annotation System and Research on Classification Model

Bachelor's Thesis | Yangzhou University | 2026

This research focuses on developing an intelligent ECG signal annotation and classification system for assisting cardiovascular disease analysis. The project combines software engineering, biomedical signal processing, and deep learning to create a complete pipeline from data annotation to automatic ECG quality classification.

Deep Learning PyTorch PyQt5 1D ResNet50 Biomedical AI ECG Signal Processing Healthcare Technology

Research Objectives

  • Develop a visual ECG signal annotation platform for expert labeling.
  • Construct a high-quality ECG signal dataset.
  • Adapt ResNet50 architecture for one-dimensional ECG signals.
  • Develop an automatic ECG quality classification model.
  • Improve intelligent healthcare analysis through AI-driven automation.

Key Contributions

  • Designed and implemented a complete ECG annotation system using PyQt5.
  • Constructed a dataset containing 11,983 annotated ECG samples.
  • Processed ECG segments consisting of 5,000 data points per sample.
  • Applied Z-score normalization for signal standardization.
  • Developed a customized 1D ResNet50 classification model.
  • Implemented AdamW optimization and Cosine Annealing learning-rate scheduling.
